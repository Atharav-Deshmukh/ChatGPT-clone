# ChatGPT Clone

A full-stack ChatGPT clone built to replicate the conversational AI experience. This project features a React frontend and a Node.js/Express backend integrated with the OpenAI API.

## 🚀 Features

* **Interactive UI:** A responsive chat interface mimicking the real ChatGPT experience.
* **AI Integration:** Connects seamlessly to the OpenAI API to generate intelligent responses.
* **Conversation Threads:** Backend models set up to handle and store chat history.
* **Full-Stack Architecture:** Clean separation of concerns between the React frontend and Node.js backend.

## 🛠️ Tech Stack

**Front-End:**
* React (with Vite)
* CSS

**Back-End:**
* Node.js
* Express.js
* MongoDB (Mongoose)
* OpenAI API
